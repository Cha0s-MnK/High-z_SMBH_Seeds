# Juodzbalis+2026 Fig.4 Observational Data

This directory stores source material, provenance notes, digitisation records, and the processed plotting CSV used to overlay the observational points from Fig.4 of Juodzbalis+2026 in `plot_Kong+2026.py`.

The processed CSV `juodzbalis2026_fig4_mbh_mstar_points.csv` contains only the points visible in Juodzbalis+2026 Fig.4. QSO1 uses the textual conservative stellar-mass upper limit `Mstar < 2e7 M_sun` and the direct MOKA3D mass `logMBH = 7.7 +/- 0.3`. The other comparison rows use marker centres digitised from the rendered source figure because the Juodzbalis+2026 source package does not include a machine-readable Fig.4 plotting table.

The figure source package is stored in `source/`, downloaded reference PDFs and PDF-to-text outputs are stored in `raw/`, and rendered/digitised products are stored in `digitisation/`.
